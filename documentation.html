QryptoCore Documentation

Overview

QryptoCore is a comprehensive encryption/decryption tool that supports multiple classical and modern cipher algorithms. The application provides a user-friendly interface for securing sensitive information using various cryptographic techniques.

Important: This documentation file is protected against modifications. Any attempt to edit or copy content will be prevented.

Features

  • Multiple cipher algorithms supported
  • Modern GUI with theme customization
  • Encryption/decryption history tracking
  • File operations (open/save)
  • Key generation for supported ciphers
  • Responsive design with adjustable font sizes
  • Three visual themes (Dark, Light, Tech)
  • Splash screen on startup
  • Status bar with operation feedback

Supported Ciphers

QryptoCore implements the following encryption algorithms:

Symmetric Key Ciphers

  • AES (Advanced Encryption Standard) - 256-bit key
  • DES3 (Triple Data Encryption Standard) - 168-bit key

Classical Ciphers

  • Caesar Cipher - Simple substitution cipher
  • VigenÃ¨re Cipher - Polyalphabetic substitution
  • One-Time Pad - Unbreakable when used correctly
  • Atbash Cipher - Hebrew substitution cipher
  • Rail Fence Cipher - Transposition cipher

Usage

To use QryptoCore:

  1. Select the appropriate tab (Encrypt/Decrypt)
  2. Enter your text or load from a file
  3. Select the cipher algorithm
  4. Enter or generate an encryption key
  5. Click the Encrypt/Decrypt button
  6. View results and optionally save to file

Technical Details

The application is built with:

  • Python 3.x
  • Tkinter for the GUI
  • PyCryptodome for cryptographic functions
  • Pillow for image handling

Key Generation

Keys are generated differently for each cipher:

- AES: 32-byte random key (Base64 encoded)
- DES3: 24-byte random key (Base64 encoded)
- Caesar: Random integer between 1-25
- VigenÃ¨re/OTP: Random 16-character lowercase string
- Atbash/Rail Fence: No key required
