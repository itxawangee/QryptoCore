QryptoCore User Manual

Classical to Quantum—One Tool to Lock Them All

Introduction

QryptoCore is a comprehensive encryption/decryption tool that supports multiple classical cipher algorithms. This user manual provides detailed instructions on how to use all features of the application.

System Requirements

  • Windows, macOS, or Linux operating system
  • Python 3.7 or later
  • Minimum 2GB RAM
  • 100MB disk space

Getting Started

Installation

  1. Download the QryptoCore package
  2. Install required dependencies: pip install -r requirements.txt
  3. Run the application: python qryptocore.py

Note: On first launch, QryptoCore will create an "assets" folder to store your settings and history.

Application Interface

The main interface consists of four tabs:

  1. Encrypt: For encrypting plaintext
  2. Decrypt: For decrypting ciphertext
  3. Settings: For configuring application preferences
  4. History: For viewing past operations

Encryption

To encrypt text using QryptoCore:

  1. Navigate to the Encrypt tab
  2. Enter or paste your plaintext in the input area
  3. Select a cipher algorithm from the dropdown
  4. Enter an encryption key (or click "Generate Key" for supported ciphers)
  5. Click the Encrypt button
  6. The encrypted result will appear in the Decrypt tab's input area

Important: Always keep your encryption keys secure. Without the correct key, your data cannot be decrypted.

Supported Ciphers

Cipher Key Requirements Description
AES Any string (hashed to 16 bytes) Advanced Encryption Standard - symmetric block cipher
Caesar Number (1-25) Shift cipher where each letter is shifted by key positions
Vigenère Text string Uses a keyword to shift letters variably
OTP Text string (must be at least as long as plaintext) One-time pad - theoretically unbreakable when used correctly
Atbash None Simple substitution cipher reversing the alphabet
Rail Fence Number (2+) Transposition cipher writing text in zigzag pattern
DES3 Any string (hashed to 24 bytes) Triple Data Encryption Standard - symmetric block cipher

Decryption

To decrypt text using QryptoCore:

  1. Navigate to the Decrypt tab
  2. Enter or paste your ciphertext in the input area (or click "Browse File" to load from file)
  3. Select the cipher algorithm that was used for encryption
  4. Enter the encryption key used originally
  5. Click the Decrypt button
  6. The decrypted plaintext will appear in the same input area

Note: You must know both the cipher algorithm and exact key used for encryption to successfully decrypt.

Troubleshooting Decryption

  • Incorrect key: Verify the key is exactly the same as used for encryption
  • Wrong cipher: Ensure you've selected the correct cipher algorithm
  • Corrupted data: If loading from file, verify the file wasn't modified
  • Format issues: Some ciphers require exact formatting (e.g., Base64 for AES/DES3)

File Operations

Loading Files

  1. In the Decrypt tab, click Browse File
  2. Select a text file containing ciphertext
  3. The file contents will load into the input area

Saving Files

  1. After encryption/decryption, ensure the result is displayed
  2. Click File → Save As... from the menu
  3. Choose a location and filename
  4. Click Save

Tip: Use the New menu option to clear all input fields when starting a new operation.

Settings & Customization

Appearance

QryptoCore offers three visual themes:

  • Dark: Dark background with light text (default)
  • Light: Light background with dark text
  • Tech: High-contrast theme with neon elements

Font Size

Adjust the font size using the slider in the Settings tab (8-18pt range).

Resetting to Defaults

Click Reset to Defaults in the Settings tab to restore all preferences to their original values.

Operation History

The History tab maintains a log of all encryption and decryption operations performed during the current session.

Features

  • View timestamp, operation type, and cipher used
  • Select an item and click Copy Selected to copy to clipboard
  • Click Clear History to remove all entries

Note: History is not preserved between application sessions.

Troubleshooting

Common Issues

Issue Possible Cause Solution
Decryption fails Incorrect key or cipher Verify both key and cipher match encryption settings
Invalid key error Wrong key format for cipher Check key requirements for selected cipher
File won't load File too large or corrupted Try a smaller file or verify file integrity
Application crashes Missing dependencies Reinstall requirements: pip install -r requirements.txt
